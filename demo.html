Cara Menggunakan

1. Include Script

<script src="script.js"></script>

2. Siapkan Container

<div id="player-wrapper"></div>

3. Inisialisasi VideoPlayer

const player = new VideoPlayer({
    wrapperId: 'player-wrapper',
    section: {
        title: 'Judul Section',
        description: 'Deskripsi section'
    },
    mainVideo: {
        url: 'https://example.com/video.mp4',
        title: 'Judul Video',
        description: 'Deskripsi video',
        type: 'video/mp4'
    },
    adVideo: {
        url: 'https://example.com/ad.mp4',
        skipAfter: 5,
        type: 'video/mp4'
    },
    thumbnailUrl: 'https://example.com/thumbnail.mp4',
    skipBackwardSeconds: 10,
    skipForwardSeconds: 10,
    infoText: 'Teks informasi kustom',
    // Custom color untuk button ads (opsional)
    adButtonColor: {
        background: '#4ade80',      // Hijau
        backgroundHover: '#22c55e',
        text: '#000000'
    },
    // Custom color untuk play button (opsional)
    playButtonColor: {
        background: '#3b82f6',      // Biru
        backgroundHover: '#2563eb',
        text: '#ffffff'
    }
});

Custom Button Colors

Kustomisasi warna button untuk play button dan skip button iklan sesuai dengan tema brand Anda.

Ad Button Color

Kustomisasi warna untuk tombol Skip Ad yang muncul saat iklan diputar.

adButtonColor: {
    background: '#4ade80',        // Warna background
    backgroundHover: '#22c55e',   // Warna saat hover
    text: '#000000'               // Warna text
}

Play Button Color

Kustomisasi warna untuk tombol Play utama yang muncul di awal.

Public Methods

player.play()

Memutar video utama

player.pause()

Menghentikan video sementara

player.setVolume(volume)

Mengatur volume (0-1). Contoh: player.setVolume(0.5)

player.seekTo(seconds)

Loncat ke waktu tertentu (dalam detik). Contoh: player.seekTo(60)

player.getCurrentTime()

Mendapatkan waktu playback saat ini

player.getDuration()

Mendapatkan durasi total video

player.destroy()

Membersihkan dan menghancurkan instance player
