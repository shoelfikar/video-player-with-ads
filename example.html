Video akan memutar iklan terlebih dahulu. Klik tombol Skip setelah 5 detik atau tunggu hingga iklan selesai.

Code:

const player = new VideoPlayer({
    containerId: 'player-1',
    mainVideo: {
        url: 'https://example.com/main-video.mp4',
        title: 'My Amazing Video',
        description: 'This is a great video about something interesting.',
        type: 'video/mp4'
    },
    adVideo: {
        url: 'https://example.com/ad-video.mp4',
        skipAfter: 5,
        type: 'video/mp4'
    },
    thumbnailUrl: 'https://example.com/thumbnail.mp4',
    skipBackwardSeconds: 10,
    skipForwardSeconds: 10,
    autoHideControlsDelay: 3000,
    infoText: 'Custom info text here'
});

Available Configuration Options:

containerId (required): ID of the container element

mainVideo.url: URL of the main video

mainVideo.title: Title of the video

mainVideo.description: Description of the video

mainVideo.type: Video MIME type (default: 'video/mp4')

adVideo.url: URL of the ad video

adVideo.skipAfter: Seconds before skip button appears (default: 5)

adVideo.type: Ad video MIME type (default: 'video/mp4')

thumbnailUrl: URL for thumbnail video

skipBackwardSeconds: Seconds to skip backward (default: 10)

skipForwardSeconds: Seconds to skip forward (default: 10)

autoHideControlsDelay: Milliseconds before hiding controls (default: 3000)

infoText: Custom info text below player

Available Methods:

player.play(): Play the video

player.pause(): Pause the video

player.setVolume(volume): Set volume (0-1)

player.seekTo(seconds): Seek to specific time

player.getCurrentTime(): Get current playback time

player.getDuration(): Get video duration

player.destroy(): Cleanup and destroy player instance
